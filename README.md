## ALGORITHM

  Open notepad and type the HTML code.

### STEP-2
  Insert the college logo using ```<img>``` tag.

### STEP-3
  Use the ```<table>``` tag with proper cell spacing and cell padding.

  Enter the slot times and days as table header using ```<th>``` tag.
